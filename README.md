# Git Basics

## Git CLI and Remote

- `git init`
- `git status`
- `git add <filename>` / `git add . `
- `git commit -m "Commit message"`
- `git push -u origin main` / `git push`
- `git remote -v`
- `git remote add origin <ssh link>`

## Git Branches & PRs

- `git switch -c <branch-name>`
- `git switch <branch-name>`
- `git switch -`
- `git branch`
- `git branch -d <branch-name>`
